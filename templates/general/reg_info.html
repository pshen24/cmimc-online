How to Register

All students will need an account on this website, which they use to submit their solutions on the day of the contest. Only one person (either a student or coach) needs to create the team.

For Students

  1. Create an account and make sure to select 'Contestant' when signing up.
  2. To join an existing team, use their invite link (then skip to step 6). To start your own team, follow steps 3-5.
  3. Go to {{ request.scheme }}://{{ request.get_host }}{% url 'contest_list' %} and select the contest you want to register for.
  4. Choose a team name.
  5. Invite other members to your team with the given invite link.
  6. Make sure each team member fills out the waiver and submits it via this form before February 23rd.

For Coaches

  1. Create an account and make sure to select 'Coach' when signing up.
  2. Go to {{ request.scheme }}://{{ request.get_host }}{% url 'contest_list' %} and select the contest you want to register for.
  3. Choose a team name
  4. Invite students to your team with the given invite link.
  5. Repeat steps 2-4 for each team you want to create.
  6. Make sure each team member fills out the waiver and submits it via this form before February 23rd.
